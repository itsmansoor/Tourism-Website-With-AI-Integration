# Tourism Website with AI Integration

## 🌍 Overview

Tourism Website with AI Integration is a full-stack web application developed as a Final Year Project. The platform helps travelers discover destinations, explore tour packages, receive AI-powered travel assistance, view real-time weather information, and book trips online through an intuitive user interface.

---

## 🚀 Features

### User Features

* User Registration & Login
* Secure Authentication
* Browse Tourism Packages
* Package Details with Image Gallery
* Real-Time Weather Information
* AI Travel Assistant
* Package Booking System
* User Profile Management
* Ratings & Reviews

### Admin Features

* Dashboard Analytics
* Manage Tour Packages
* Manage Users
* Manage Bookings
* View Customer Reviews
* Package Performance Tracking

### AI Features

* AI-powered Travel Recommendations
* Destination Suggestions
* Travel Planning Assistance

---

## 🛠️ Technologies Used

### Frontend

* React.js
* React Router
* Redux Toolkit
* Tailwind CSS
* Swiper.js

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Multer

### APIs & Integrations

* OpenRouter AI API
* OpenWeather API
* OpenCage Geocoding API
* Google Maps API
