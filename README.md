# Tourism Website with AI Integration

Final Year Project

## Features

- AI Travel Recommendations
- Tourism Packages
- Booking System
- Weather Information
- User Authentication
- Stripe Payments
- Reviews & Ratings
- Admin Dashboard
- Responsive Design

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Redux Toolkit

Backend:
- Node.js
- Express.js
- MongoDB

AI:
- Google Gemini API

Deployment:
- Vercel
- Render
